How to use cutting edge version of dotnet overlay from github?

 Gentoo Mono Handbook
 
Add a line
https://raw.githubusercontent.com/ArsenShnurkov/shnurise/master/github-gentoo-dotnet-metadata-for-layman.xml
into "overlays:" section of /etc/layman/layman.cfg.

This will allow you not to wait the synchronization with sync gentoo.org
