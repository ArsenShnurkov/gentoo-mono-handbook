 Gentoo Mono Handbook
 
man mdtool
mdtool is a tool that can execute head-less MonoDevelop applications, this can be used for example to do batch compilations without starting up the MonoDevelop GUI.
Use this tool to build any solution supported by MonoDevelop (eg. .mds, .sln). If no solution/project is specified then it will pull the first solution it can find in the current directory. You can use this option to build packages in a head-less or batch fashion.
